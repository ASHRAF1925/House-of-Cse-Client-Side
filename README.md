# House Of CSE



House of CSE is a educational website where you can learn about different courses of Computer Science and Engineering.

## Features

- Different Types of Courses
- Secure login
- Register options
- Data loaded from API
- Attractive UI
- Build with new technologies etc.

## Technology,Libraries and Language Used

- HTML
- CSS
- JAVA SCRIPT
- Bootstrap
- React.js
- Node.js

## Login System Implemented

- Log in with Email and Password
- Log in with Google Popup
- Log in with Github
